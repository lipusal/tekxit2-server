Recommendations:
-Disable/Remove ProjectE for multiplayer server, it can cause many issues in the long run.
-Keep taint disabled in the Thaumcraft config.
~More recommendations may be added over time so make sure to check this every update.~

To start the server open StartTekxitServer.bat

The default memory that is given to the server is 3GB, to increase or decrease this right click StartTekxitServer.bat and click edit then change both instances of "3G" to whatever amount of memory you want to allocate.
For 2 GB of memory EXAMPLE: java -Xmx2G -Xms2G -jar forge.jar nogui
For 4 GB of memory EXAMPLE: java -Xmx4G -Xms4G -jar forge.jar nogui
